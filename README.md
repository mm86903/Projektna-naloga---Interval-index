# Interval index 

## Uvod
V tej projektni nalogi smo analizirali intervalni indeks grafov, ki meri, koliko vozlišč je vključenih na najkrajših poteh med pari vozlišč. Višja kot je vrednost intervalnega indeksa, bolj so sosednja vozlišča oddaljena.

Analizo smo izvedli za:
- Poti $P_n$, da preverimo, ali maksimirajo intervalni indeks.
- Kubične (3-regularne) grafe, da poiščemo minimalne in maksimalne vrednosti indeksa.
